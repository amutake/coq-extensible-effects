[WIP] coq-extensible-effects
============================

Extensible Effects in Coq.

[Freer Monads, More Extensible Effects](http://okmij.org/ftp/Haskell/extensible/)

Requirements
------------

- Coq 8.4pl6
- Ssreflect 1.5

Install
-------

```sh
cd /path/to/coq-extensible-effects
./configure
make
make install
```
